Mars Rover Kata

  1. Create obstacles (optional)
  2. Set X and Y. Take values [0-9]. 0 and 9 included
  3. Make string of commands.
    • Use l- to turn rover left
    • Use r- to turn rover right
    • With f- move it forward
    • With b - move it back
    • You can see rover's moves on grid and also in console.
